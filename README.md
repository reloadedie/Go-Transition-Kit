# GoTransition Kit 

GoTransition Kit — это комплект методических материалов для DevOps инженеров, 
Team Lead-ов и архитекторов, которые планируют перевести существующие 
микросервисы или монолитные модули на Golang.

В проекте нет знаний синтаксиса Go, здесь готовые стратегии миграции, 
шаблоны рефакторинга и инструменты оценки, которые позволяют заменить 
устаревшие модули на быстрые Go-сервисы с минимальными трудозатратами 
и без остановки продакшена

# Целевая аудитория
CTO и Tech Lead-ы, ищущие пути оптимизации производительности.

Разработчики, которым нужно "легаси" переписать на Go, но непонятно, с чего начать.

Команды, которые хотят внедрить Go для оптимизации, но боятся долгого простоя
